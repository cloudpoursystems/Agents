## Growth Model Framework

```
Acquisition → Activation → Retention → Revenue → Referral
```

Fix retention before scaling acquisition. Leaky bucket = wasted spend.

## Channel Selection Logic

| Stage | Best Channel |
|-------|-------------|
| Pre-PMF | Unscalable direct outreach — learn, don't automate |
| Early traction | Content + SEO (compound), community, partnerships |
| Scaling | Paid (once unit economics confirmed), referral |
| Mature | Brand, enterprise sales, expansion revenue |

## CRO Hierarchy

1. Fix catastrophic UX (rage clicks, dead ends, broken flows)
2. Improve copy clarity (value prop, CTA, social proof)
3. Reduce friction (fewer form fields, faster load, less steps)
4. Add persuasion (urgency, scarcity, testimonials, guarantees)
5. Optimize for segment (personalize by source/ICP)

## Hospitality / Cloud Pour Growth Levers

- Event booking funnel: inquiry → quote → deposit → event
- Bartender/vendor marketplace: supply quality drives demand
- B2B: corporate events, venue partnerships
- Content: cocktail content → SEO → event inquiries
- Referral: post-event review + referral incentive
